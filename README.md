
You need to install:

numpy

pytorch (try to get rid of it soon)

waymo-development-kit

You need to download the dataset from Waymo Open Dataset (Warning: about 1TB)  and the folder structure is like this:

```
waymo
├── raw_data
│   ├── segment-....
├── LICENSE
```
